# transaction-note

## Getting Started 

1. Clone the repository.
```
git clone https://github.com/vertices-network/transaction-note.git
```

2. Install the dependencies.
```
npm i
```

3. Test the project.
```
$ node note.js
```

**You should have the Transaction Id returned if everything went well.**
```

```
